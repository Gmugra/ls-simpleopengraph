INSTALLATION
1. Rename the folder with the plugin code, it must has name "simpleopengraph".
2. Copy the plugin in /plugins/ directory.
3. Using plugins managment (/admin/plugins/) to activate it.

Плагин "Simple Open Graph" для LiveStreet CMS 1.0.X
Текущая версия плагина: 1.33

Плагин разработан для и используется на сайте ММОзговед (http://mmozg.net).

Плагин добавляет мета-данные Open Graph: http://ogp.me/
Подробно: http://livestreet.ru/blog/addons/18947.html

УСТАНОВКА:
1. Папку с кодом плагина обязательно переименовать в "simpleopengraph".
2. Скопировать плагин в каталог /plugins/
3. Через панель управления плагинами (/admin/plugins/) запустить его активацию.
